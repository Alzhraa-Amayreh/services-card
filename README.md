# SmartWeb Services
This project presents SmartWeb digital services through six service cards. Each card highlights a different service, including web design, SEO optimization, social media design, landing page development, UI improvement, and technical support. The cards provide a simple overview of the services offered by SmartWeb in a clean and organized layout.

## Technologies Used (HTML & CSS)
The project was built using HTML and CSS. HTML was used to create the page structure and organize the service cards, while CSS was used for styling and layout. CSS Grid was used to arrange the six service cards in a clean and responsive layout. CSS variables were used to store colors and reduce code repetition. Border-radius was applied to create smooth card shapes, and box-shadow was used to improve the visual 
appearance. Hover effects and transitions were added to create simple interactive visual effects when users interact with the cards.

## Page Sections
The page contains a simple services section that includes a main title, a short description, and a grid of six service cards. Each card presents a SmartWeb service with an icon, service title, short description, and a button. The layout is organized using CSS Grid to provide a clean and structured design.
